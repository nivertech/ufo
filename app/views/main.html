What?

Look up UFO sightings reported to the National UFO Reporting Center throught history, or view sightings near your location.

How?

I developed an API in Ruby on Rails to scrape the NUFORC website and then return the sightings in JSON.

API's Used

  • NUFORC API
  • WeatherUnderground Geolookup API
  • HTML5 GeoLocation API

HTML5 Technology

In my location service, in app/scripts/services/locationservice.js, you'll see I am using the HTML5 GeoLocation API to obtain the user's location. I grab their latitude and longitude which I then feed into the WeatherUnderground Geolookup API. That returns a city and state which I then can use with my NUFORC API to look for reported sightings in that location.

Extras

I used AngularJS to structure my application. Angular follows an MVC like pattern that breaks up different components of your application into modules. I wrote some custom services in Angular that follow the singleton pattern.

The custom services in app/scripts/services, filters in app/scripts/filters, and the controllers in app/scripts/controllers are the JavaScript I wrote. I got the filters and location service working with some help from Stack Overflow

Design

It is designed for desktop use, but since I am using Twitter Bootstrap I could make some tweaks to make it more mobile friendly in the future.

I wanted to keep it very clean and simple so you can easily read the information that is presented. I thought about a dark theme because of the theme of the website, I thought it might fit, but I think dark text on light backgrounds is much more legible.

For the 'Past Sightings' page I was hoping to implement a calendar type view where you could enter a year then page through the months. It would show how many sightings were reported on each day in the calendar then you could click on a day to see more details. Unfortunately, I did not have enough time for this. Instead you can enter a specific date in history and view any sightings reported that day.
